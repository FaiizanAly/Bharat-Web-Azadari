## 🌟 About

Bharat Web Azadari is a digital platform dedicated to covering and sharing Shia Azadari events, including:

* Live Majlis
* Live Nohay
* Live Programs
* Juloos Coverage
* Muharram Events
* Religious Gatherings
* Community Announcements

The platform provides easy access to event schedules, live streams, videos, announcements, and gallery content for the Azadari community.
